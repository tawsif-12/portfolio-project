# Public Assets

Place your public assets in these folders:

## Recommended Structure

```
public/
├── avatar.jpg              # Your profile picture
├── resume.pdf             # Your resume PDF
├── vite.svg               # Favicon (replace with your own)
├── projects/              # Project screenshots
│   ├── ecommerce-1.jpg
│   ├── ecommerce-2.jpg
│   └── ...
└── testimonials/          # Testimonial photos
    ├── sarah.jpg
    ├── michael.jpg
    └── ...
```

## Image Guidelines

- **Avatar**: 400x400px, square format
- **Project Screenshots**: 1200x600px recommended
- **Testimonial Photos**: 200x200px, square format

## Placeholders

If you don't have images yet, the site will show placeholder content. Replace with your actual images when ready.
